# Frontend Style Guide

> Murmur 前端视觉规范。复古终端风 + 现代可读性。

## Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| `--terminal-bg` | `#1a1a2e` | 主背景 |
| `--terminal-bg-secondary` | `#16213e` | 卡片/面板背景 |
| `--terminal-amber` | `#ffb000` | 主文字色（琥珀主题） |
| `--terminal-green` | `#00ff41` | 主文字色（绿色主题） |
| `--terminal-text-dim` | `#996a00` | 次要文字 |

## Character Colors

每个角色分配独立颜色：
- `--char-color-1`: `#ff6b6b` (红)
- `--char-color-2`: `#4ecdc4` (青)
- `--char-color-3`: `#ffe66d` (黄)
- `--char-color-4`: `#a29bfe` (紫)
- `--char-color-5`: `#fd79a8` (粉)

## Typography

- 主字体：JetBrains Mono / Fira Code（等宽）
- 旁白字体：Noto Serif SC（衬线）
- 打字机效果：逐字显示 + 光标闪烁

## Effects

- 文字光晕（text-shadow）
- 扫描线（scanlines，subtle）
- 光标闪烁（cursor blink animation）
